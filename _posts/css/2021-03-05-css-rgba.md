### 문제점
---
배경색을 흐리게 넣고 싶은데 opacity를 활용할 경우 다른 구성요소들도 흐리게 표현되는 문제가 있었다.

### 해결방법
---

```css
바꾸고 싶은 Element { background: rgba(RGB값, 투명도); }

.id { background: rgba(255, 102, 81, 0.5); }    //예시
```

